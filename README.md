
# Bees App

Bees is a social media android application which aims at making the social media life of individuals much more data-safe, secure and interactive!

## Features

- Verified User Login - Each account is verified through e-mail verification before creating an account.
- Blogging - Users are provided with the option of creating/editing/deleting micro and mini blogs with an optional image upload.
- NSFW(Not Safe For Work) Detection - Each new post uploaded by the user goes through a filter, which restricts inappropriate images from begin shared in the application.
- Block/Reporting Posts and Users - Users can also block/report posts/users whichever they find inappropriate.
- Profile Edit - View/Edit self profile details.
- Unblock Users - Unblock previously blocked users.

## Features (Tech)
- Firebase onEventListener Socket API used for Realtime Feeds upadates.
- Clean Architecture (on MVVM) used.
- Dagger-Hilt Dependency Injection used.
